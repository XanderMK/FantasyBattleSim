# Fantasy Battle Simulator
A turn-based battle simulator created for our Programming I final project, built off Jaylib.
## Gameplay Features
- Turn-based battle mechanics
  - Character and monster interaction
  - Health, defense, and mana system
  - Various items
  - Inventory management
  - Normal attack & magic attack
  - Critical attacks & miss attacks
  - XP
- Items
  - Health Potion (+10 HP)
  - Mana Potion (Fully restores mana)
  - Fireball (Deals 15 damage to a monster)
## Other Features
- Sound effects
- Scene management
- Dialogue boxes
- Unity style component management
## Running & Compiling
You can easily run and compile the game yourself with the following steps:
1. Install IntelliJ IDEA or another Java IDE
2. Copy this repository's URL and use IntelliJ's "Clone Repository" feature
3. Click "Clone"
4. IntelliJ may prompt you to install JDK 17, if so, install it
5. Navigate to `Main.java` in `src/main/java/main/Main.java`
6. Press "Run"
